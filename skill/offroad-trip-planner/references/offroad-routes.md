# 越野路线知识库

## 轻度越野定义

轻度越野指在安全前提下行驶非铺装路面（林道、山道、土路、碎石路），不涉及：
- 陡坡攀爬（>30度）
- 涉水深度超过排气管
- 泥沼/沙地陷车风险路段
- 需要绞盘/拖车救援的路段

适合车型：SUV、越野车。轿车应只走铺装路面。

## 典型越野路线参考

### 东北方向（北京出发）
- 北京 -> 承德 -> 赤峰 -> 通辽 -> 长春 -> 哈尔滨 -> 齐齐哈尔 -> 漠河
- 特色：大兴安岭林海、边境风光、冬季雪原
- 越野路段：大兴安岭林区林道、边境公路

### 西北方向（北京出发）
- 北京 -> 呼和浩特 -> 额济纳旗 -> 哈密 -> 乌鲁木齐 -> 伊犁
- 特色：草原、戈壁、胡杨林、天山
- 越野路段：草原土路、戈壁碎石路

### 西南方向（北京出发）
- 北京 -> 西安 -> 成都 -> 康定 -> 稻城亚丁
- 特色：川藏线、雪山、藏族文化
- 越野路段：高原山路、非铺装山路

## 天气对越野的影响

| 天气 | 影响 | 建议 |
|------|------|------|
| 大雨 | 非铺装路泥泞打滑 | 避开越野路段，走铺装公路 |
| 中雨 | 路面湿滑 | 谨慎驾驶，减速 |
| 雪/冰 | 轮胎抓地力下降 | 雪地胎/防滑链，越野路段绕行 |
| 雾 | 能见度低 | 山路减速，开雾灯 |
| 晴天 | 路况最佳 | 适合所有越野路段 |

## 越野装备清单

- 必备：备胎、拖车绳、千斤顶、急救包
- 推荐：充气泵、对讲机、车载灭火器
- 进阶：防滑链（冬季）、脱困板、绞盘
